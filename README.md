Sistema de Controle de Clientes

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação e orientação a objetos.  
A aplicação realiza o cadastro de clientes e calcula o valor total de uma compra com base no tipo de cliente.

## Funcionalidades

- Cadastro de clientes
- Cálculo de imposto sobre o valor da compra
- Diferenciação entre pessoa física e pessoa jurídica
- Exibição do valor da compra, imposto e total a pagar

## Regras de negócio

- Pessoa Física: 10% de imposto
- Pessoa Jurídica: 20% de imposto

## Tecnologias utilizadas

Versão web:
- HTML
- CSS
- JavaScript

Versão console:
- C#
- .NET

## Estrutura do projeto

Versão web:
- index.html: interface da aplicação
- style.css: estilização
- app.js: lógica de cálculo

Versão C#:
- Program.cs: ponto de entrada do sistema
- Clientes.cs: classe base
- Pessoa_Fisica.cs: regras para pessoa física
- Pessoa_Juridica.cs: regras para pessoa jurídica

## Como executar

Opção 1 (navegador):
- Abra o arquivo index.html no navegador
- Preencha os dados e execute o cálculo

Opção 2 (servidor local):
- Execute o comando:
  python -m http.server 5500
- Acesse: http://localhost:5500

Opção 3 (C#):
- Execute o comando:
  dotnet run
